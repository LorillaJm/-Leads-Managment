# Dashboard User Guide

## Quick Start

The new dashboard provides a comprehensive view of your sales performance with real-time metrics, filtering capabilities, and detailed analytics.

## Dashboard Sections

### 1. Header Section
Located at the top of the page:
- **BYD Logo**: Company branding
- **Dashboard Title**: "ADMIN | MANAGEMENT DASHBOARD"
- **System Labels**: Environment and template information
- **Admin Badge**: "FOR ADMIN ONLY" indicator

### 2. Filter Panel (Left Sidebar)
Control what data you see:

#### Year Filter
- Select from available years (2024, 2025, 2026)
- Changes all metrics to show data for selected year

#### Month Filter
- Check/uncheck individual months (JAN-DEC)
- Select "ALL" to view all months
- Multiple months can be selected simultaneously

#### Sales Consultant Filter
- Dropdown list of all sales consultants
- Select "ALL" to view team-wide metrics
- Select individual consultant to view their performance

### 3. KPI Panel (Left Side)
Six key performance indicators displayed vertically:

1. **LEADS** (Gray)
   - Total leads count
   - Shows goal target

2. **PROSPECTS** (Blue)
   - Qualified prospects count

3. **TEST DRIVES** (Light Blue)
   - Completed test drives

4. **RESERVATIONS** (Lighter Blue)
   - Vehicle reservations made

5. **BANK APPLICATIONS** (Yellow)
   - Financing applications submitted

6. **CLOSED DEALS** (Green)
   - Successfully closed sales

### 4. Main Content Area (Center)

#### Overview Panel
- Large numeric display of total count
- "By count" label for context

#### Conversion Flow Panel
Shows conversion rates between stages:
- **Leads → Prospects**: Percentage with goal (20%)
- **Prospects → Closed Deals**: Percentage with goal (25%)
- Visual line chart showing conversion funnel

#### Activity Breakdown Panel
Displays activity metrics with minimum targets:
- **Test Drives**: Count with minimum target (300)
- **Reservations**: Count with minimum target (120)
- **Bank Applications**: Count with minimum target (180)

#### Analytics Chart
Multi-category bar chart showing:
- Performance across all metrics
- Color-coded by category
- Top 10 sales consultants
- Interactive tooltips on hover

### 5. Sales Team Table (Right Sidebar)
Comprehensive team performance table:

#### Columns
- Sales Consultant name
- Leads count
- Prospects count
- Test Drives count
- Reservations count
- Bank Applications count
- Closed Deals count

#### Features
- **Sortable**: Click column headers to sort
- **Scrollable**: Handles large team sizes
- **Count Display**: Shows total number of consultants
- **Action Buttons**: Quick access tools in header

## How to Use

### Viewing Overall Performance
1. Set filters to "ALL" for year, months, and consultant
2. Review KPI panel for quick metrics
3. Check conversion rates in Conversion Flow panel
4. Analyze team performance in Sales Team table

### Analyzing Individual Performance
1. Select specific sales consultant from filter
2. KPI panel updates to show their metrics
3. View their position in Sales Team table
4. Compare against team averages

### Filtering by Time Period
1. Select desired year from Year dropdown
2. Check specific months or select "ALL"
3. All metrics update automatically
4. Charts and tables reflect filtered data

### Comparing Performance
1. Use Sales Team table to compare consultants
2. Sort by different metrics (click column headers)
3. Review Analytics Chart for visual comparison
4. Check conversion rates against goals

## Mobile Usage

### Accessing Filters (Mobile)
1. Tap "Open Filters" button at top
2. Adjust year, months, and consultant
3. Tap "Close Filters" to view results

### Viewing Data (Mobile)
- Scroll vertically through all sections
- KPI cards stack vertically
- Tables scroll horizontally if needed
- Charts resize to fit screen

## Tips & Best Practices

### Performance Monitoring
- Check KPI panel daily for quick overview
- Monitor conversion rates against goals
- Review Sales Team table for team insights
- Use Analytics Chart for trend analysis

### Goal Tracking
- Compare actual vs. goal in KPI cards
- Monitor conversion rate goals (20%, 25%)
- Track minimum activity targets
- Identify underperforming areas

### Team Management
- Sort Sales Team table by different metrics
- Identify top performers
- Spot consultants needing support
- Balance workload across team

### Data Analysis
- Use filters to isolate specific periods
- Compare month-over-month performance
- Analyze individual consultant trends
- Review activity breakdown for insights

## Understanding Metrics

### Leads
Total number of potential customers entered into system

### Prospects
Leads that have been qualified and show genuine interest

### Test Drives
Number of test drives scheduled and completed

### Reservations
Customers who have reserved a specific vehicle

### Bank Applications
Financing applications submitted to banks

### Closed Deals
Successfully completed sales transactions

### Conversion Rates
- **Leads → Prospects**: Qualification effectiveness
- **Prospects → Closed Deals**: Sales closing effectiveness

## Troubleshooting

### Data Not Loading
- Check internet connection
- Refresh the page
- Verify you're logged in
- Contact support if issue persists

### Filters Not Working
- Ensure at least one month is selected
- Try selecting "ALL" to reset
- Refresh page if needed

### Table Not Sorting
- Click directly on column header text
- Wait for data to load completely
- Try different column if one doesn't work

### Charts Not Displaying
- Ensure browser is up to date
- Check if data exists for selected filters
- Try different time period
